# Zakir Enterprise — Web Design System

## 1) Brand Direction
Zakir Enterprise is a Bangladesh-based construction and infrastructure company.
The website should feel:
- premium
- trustworthy
- industrial but refined
- bold and business-focused
- modern, clean, and conversion-oriented

The visual tone should balance:
- construction strength
- corporate credibility
- project showcase elegance
- strong CTA-driven business conversion

This design system is based on the existing Zakir landing-page direction and visual assets:
- lime / yellow-green brand accent from the logo
- dark charcoal overlays
- warm stone / concrete neutrals
- spacious premium section layouts
- clean sans-serif typography
- image-first construction portfolio presentation

---

## 2) Core Color System

### Primary Brand Colors
- **Brand Lime:** `#B4DE35`
- **Brand Lime Hover / Deep Lime:** `#8DAC24`
- **Dark Charcoal:** `#111111`
- **Soft Charcoal:** `#1D1F1E`

### Neutral Support Colors
- **Off White:** `#F7F7F5`
- **Warm Concrete:** `#E8E3DB`
- **Stone Gray:** `#C9C3BA`
- **Text Gray:** `#6B6F6A`
- **Border Gray:** `#D9D6CF`

### Optional Support / Utility Tones
- **Deep Olive Tint:** `#5E6E22`
- **Muted Bronze / Earth Accent:** `#A68B71`
- **Overlay Black 80:** `rgba(0,0,0,0.80)`
- **Overlay Black 60:** `rgba(0,0,0,0.60)`
- **Overlay Lime Glow:** use very subtly for focus or CTA emphasis only

### Color Usage Rules
- Use **Brand Lime** as the main action and emphasis color.
- Use **Dark Charcoal** or dark image overlays for premium construction feel.
- Use **Off White / Warm Concrete** for backgrounds to maintain a clean and sophisticated look.
- Avoid overly saturated colorful combinations beyond the core accent system.
- Limit the lime accent to important UI moments:
  - CTA buttons
  - active tabs
  - badges
  - highlight stats
  - section accents
  - hover/focus states
- The page must remain elegant and premium, not noisy.

---

## 3) Typography System

### Typography Mood
Use a modern, bold, clean sans-serif system.
The type should feel:
- strong
- readable
- premium
- editorial enough for hero sections
- highly legible for long service content

### Recommended Font Pairing
Primary choice:
- **Heading Font:** Manrope / Plus Jakarta Sans / Inter Tight
- **Body Font:** Inter / Manrope / DM Sans

Best practical pairing:
- **Headings:** Manrope
- **Body:** Inter

### Type Scale
- **Display / Hero Title:** 56–72px, bold / extra-bold
- **Page Hero Title:** 44–56px, bold / extra-bold
- **Section Title:** 32–40px, semi-bold / bold
- **Subsection Title:** 24–28px, semi-bold
- **Card Title:** 20–24px, semi-bold
- **Body Large:** 18–20px
- **Body Regular:** 16–18px
- **Small Supporting Text:** 14px
- **Caption / Meta:** 12–13px

### Typography Rules
- Use strong heading contrast.
- Keep line length comfortable for business readability.
- Do not use decorative fonts.
- Headings should have clear visual hierarchy and generous spacing.
- Body copy should feel professional and easy to scan.

---

## 4) Layout & Grid System

### Page Width
- Max content width: **1320px**
- Standard content width: **1200–1280px**

### Grid
- 12-column responsive grid
- Desktop gutter: 24px
- Tablet gutter: 20px
- Mobile gutter: 16px

### Section Spacing
- Desktop section padding: **96–120px**
- Inner block spacing: **24–40px**
- Card padding: **24–32px**
- Mobile section padding: **56–72px**

### Border Radius
- Small cards: **16px**
- Medium cards: **20px**
- Large panels / hero cards / feature containers: **24–32px**
- Buttons: **14–18px**

### Shadows
Use soft premium shadows only:
- subtle depth
- no harsh UI shadows
- layered card elevation where necessary

Suggested shadow style:
- `0 10px 30px rgba(0,0,0,0.08)`
- `0 18px 50px rgba(0,0,0,0.10)`

## 5) Visual Style Principles

### Overall Style
- premium construction portfolio website
- modern editorial sectioning
- large imagery
- layered dark overlays
- strong alignment
- minimal clutter
- high trust

### Key Visual Ingredients
- full-width hero areas
- large service-related construction photography
- dark overlay banners with strong white text
- lime-highlighted CTA buttons
- clean off-white content sections
- occasional warm concrete background blocks
- icon cards with restrained accents
- stat or trust strips
- timeline/process layouts
- premium accordion FAQ blocks

### Image Direction
Use images that show:
- building construction
- site execution
- road works
- foundation work
- heavy equipment
- workforce in action
- engineering quality
- safety and delivery discipline

Image treatment:
- cinematic and high contrast
- realistic, not stocky if possible

- keep image cropping confident and architectural

## 6) Core UI Components

### Buttons
#### Primary Button
- background: Brand Lime
- text: Dark Charcoal
- bold label
- medium-large size
- slightly rounded corners
- hover: deeper lime / slight lift

#### Secondary Button
- transparent or dark background depending on section
- border: subtle neutral or white
- text: white on dark, charcoal on light
- hover: soft fill transition

#### Ghost / Text Button
- used for secondary navigation
- arrow icon optional
- clean and modern

### Cards
Use premium cards with:
- light backgrounds on light sections
- dark cards only for emphasis sections
- ample spacing
- icon or number anchors
- small supporting divider or tag when useful

### Chips / Tags
Use for:
- service label
- project type
- trust markers
- process numbering
Style:
- muted background
- small radius
- minimal and elegant

### Accordion
FAQ accordion should be:
- premium and spacious
- bold question rows
- subtle separators
- smooth expand/collapse
- easy tap target on mobile

### Stats / Trust Cards
Can include:
- years of experience
- completed projects
- skilled workforce
- safety / compliance strength
- nationwide capability

Style:
- compact but strong
- use icons or numeric emphasis
- keep copy very short

---

## 7) Motion & Interaction

### Motion Style
- smooth
- premium
- restrained
- modern
- not flashy

### Recommended Motion
- fade + slight upward reveal on scroll
- card hover lift
- image zoom on hover
- smooth button state transitions
- FAQ expand/collapse motion
- timeline / process section reveal

### Avoid
- overscaled animations
- bright glowing effects
- playful motion
- anything that makes the brand feel less corporate

---

## 8) Iconography
Use:
- simple line icons
- modern rounded icon style
- construction/infrastructure related symbols
- no cartoon icons
- no overly colorful icons

Preferred icon tone:
- charcoal on light sections
- white on dark sections
- lime used sparingly for highlight

---

## 9) Page Background Strategy
Use alternating backgrounds for rhythm:
- Hero / CTA sections: image + dark overlay
- Main content sections: Off White
- Supporting contrast sections: Warm Concrete / very light stone
- Trust / premium CTA areas: Dark Charcoal

This alternation helps long pages feel premium and readable.

---

## 10) Design Rules for Inner Pages
All inner pages should remain visually aligned with the landing page.

### Inner Page Formula
1. strong hero banner
2. concise intro
3. structured content sections
4. trust / proof blocks
5. related project links
6. premium CTA banner
7. clean footer

### Inner Page Hero Rules
- strong background image
- dark overlay
- service title
- subtitle/value proposition
- breadcrumb optional
- CTA optional if composition allows

---

## 11) Service Detail Page-Specific UI Rules

### Service Hero
Must feel premium and high-intent.
Include:
- service title
- short subtitle / value proposition
- related construction image
- strong overlay
- optional micro trust badges

### Service Overview
Long-form but highly readable.
Use 2-column layout where useful:
- left: heading and summary
- right: full explanatory copy or quick bullets

### Scope of Work
Preferred patterns:
- icon cards
- 2-column or 3-column cards
- concise service capability statements

### Execution Process
Preferred pattern:
- horizontal step cards on desktop
- vertical timeline on tablet/mobile
- clear step numbering

### Why Choose Us / Benefits
Use:
- 3 to 6 benefit cards
- short business value-led copy
- icons
- visual consistency with landing-page brand tone

### Machinery / Method / Strengths
This can be shown as:
- stat cards
- icon blocks
- image + bullet list split section

### Related Projects
Use a premium project strip or card grid.
Each card should include:
- project image
- project title
- location or type
- small short descriptor
- CTA to view project

### FAQ
Use structured accordion with business-friendly copy.
This section should feel helpful and SEO-friendly.

### CTA Banner
Very important conversion section.
Must include:
- background image
- dark overlay
- dynamic headline based on service
- supporting text
- primary CTA
- secondary CTA

CTA style should feel premium and decisive.

---

## 12) Responsive Behavior

### Mobile Rules
- keep strong section hierarchy
- stack columns cleanly
- preserve CTA visibility
- use large tap targets
- maintain hero readability
- convert process timeline into vertical steps
- related projects become swipe cards or 1-column stack
- optional sticky bottom CTA can be used on mobile

---

## 13) Content Tone Rules
Copy should feel:
- business-first
- direct
- trust-building
- professional
- confident
- not overly technical for general visitors
- persuasive without sounding salesy

Use concise headings and benefit-oriented support copy.

---

## 14) Accessibility & UX Rules
- Maintain strong contrast on dark hero and CTA sections.
- Ensure button labels are easy to read.
- Keep body text legible with enough line-height.
- Do not overload sections with too much dense text.
- Use clear visual grouping and consistent spacing.
- Maintain clear CTA hierarchy.

---

## 15) Component Checklist for Claude / Design Generation
When generating new pages in this project, keep these constants:
- premium construction aesthetic
- lime accent + charcoal base
- warm concrete neutrals
- strong image-led sections
- spacious modern grid
- bold sans-serif headings
- refined cards and timelines
- elegant FAQ accordion
- strong CTA conversion blocks
- clean responsive behavior
